# Labeling Train Demo

This demo is for prompt engineering development

# Requirement

1. You need your own AI API & key to set in

# Run

1. install requirement

### `pip install openai`

2. set your own api_keys in `/keys/.api_keys`
3. the pattern of `.api_keys` follows `api_key_name=value`
4. modify the parameter name of `qwen_key = api_keys.get('QWEN_API_KEY')` as your own in `.api_keys`
5. run it

### `python main.py`
